## Формат работ
- В каждой папке есть `solution.py` (скрипт) и `*.ipynb` (ноутбук для сдачи).
- Для удобства проверки рекомендуется использовать именно ноутбуки.

## Запуск
```bash
source .venv/bin/activate
python tmo_variant_6/lab_01_eda/solution.py
```

Пример запуска ноутбука:
```bash
source .venv/bin/activate
jupyter notebook tmo_variant_6/lab_01_eda/lab_01.ipynb
```

Для каждой работы результаты и графики сохраняются в `artifacts/` внутри соответствующей папки.
